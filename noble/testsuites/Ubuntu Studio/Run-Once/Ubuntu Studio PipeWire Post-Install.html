Audio Check Before you begin, make sure you are booted into a fresh session, and that you haven't opened any audio applications - including any browser pages with audio content.
Open Patchance from the taskbar and keep it around for your patchbay.
Open Hydrogen from the menu
Create a quick beat and play it - you should hear sound. If you aren't hearing any sound, make sure that connections are accurate in Patchance, and that your hardware levels are not muted.
Make sure Hydrogen is connected to your speakers
In Patchance make sure that Hydrogen exists and is connected to system playback
You can close Hydrogen once you are done
Test desktop sound through Pipewire
Open firefox and go to any page that has audio. Flash video will not work, so html5 is preferred - such as http://youtube.com/html5.
